# Butterfly Species Classification using CNN

A Convolutional Neural Network (CNN) model built using TensorFlow and Keras to classify different butterfly species.

## Dataset
The dataset contains labeled images of butterfly species. Due to size limits, the raw images (`train/`, `test/`) and CSV metadata (`Training_set.csv`, `Testing_set.csv`) are excluded from this repository.

## Model
The project uses a custom Convolutional Neural Network (CNN) with Conv2D, MaxPooling2D, Dropout, and Dense layers.

## Setup & Requirements
To run the notebook, install the dependencies:
```bash
pip install tensorflow numpy pandas matplotlib scikit-learn
```
